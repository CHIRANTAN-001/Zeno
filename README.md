# 🚀 Zeno - Search Engine from Scratch in Go

Building a search engine from scratch to understand how real-world systems like Elasticsearch and Google Search work under the hood.

---

## 🧠 What This Project Covers

- Tokenization pipeline
- Stopword filtering
- Inverted Index (in-memory)
- Query processing
- Structured search results
